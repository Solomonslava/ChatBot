QAP_bot
Телеграм-бот для конвертации валют. Имя в телеграме @Convert_crypt_Bot

Этот телеграм-бот выполняет несколько команд:

/start выдает стартовое сообщение и пишет, какие команды еще он умеет выполнять;
/help напоминает, какие команды умеет выполнять бот;
/values выводит список доступных для конвертации валют;
<валюта 1> <валюта 2> <количество> выводит значение количества валюты 2 от валюты 1.
Данный бот не запущен, необходимо запустить бот со своего устройства (загрузить данный репозиторий на свой компьютер)
Для работы необходимо установить пакеты requests и PyTelegramBotAPI.
Бот написан на Python версии 3.10.
